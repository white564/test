# 基本コマンド

$ git init

gitの管理を始める


$ git status


いまのファイルの状態を確認する


$ git log

コミットの履歴を参照する(歴史を見る)


$ git diff

コミットとの差分を確認する


$ git add -A

コミット対象にするファイルを追加する(staging)


$ git commit -m “コミットメッセージ”

メッセージ付きで現在のファイルの状況を保存する(commit)


$ git push

githubなどのリモートリポジトリに現在のファイルの状況を送る


$ git pull

githubなどのリモートリポジトリからファイルの状況を取ってきて適用する
